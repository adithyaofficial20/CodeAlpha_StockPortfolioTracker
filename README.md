# Stock Portfolio Tracker

## Description

A Python-based application that helps users track stock investments, calculate portfolio value, and analyze profit or loss.

---

## Features

- Add Stocks
- View Portfolio
- Investment Calculation
- Current Market Value
- Profit/Loss Analysis
- CSV Export
- Portfolio Loading

---

## Technologies Used

- Python
- CSV Module
- Dictionaries
- Functions

---

## How To Run

```bash
python main.py
```
